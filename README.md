# gitlab-lint Frontend

![gitlab-lint Frontend](gitlab-lint-frontend.png)

## Dependencies

- nodejs
- yarn
- [gitlab-lint API and collector][gitlab-lint]

### Installing dependencies

```bash
make setup
```

## Run it

```bash
make run
```

[gitlab-lint]: https://github.com/globocom/gitlab-lint
